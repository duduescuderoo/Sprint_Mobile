## Objetivo
Desenvolver uma aplicação mobile para auxiliar a Ford na retenção de clientes no pós-venda,
permitindo acompanhamento de revisões, geração de leads inteligentes e alertas personalizados.

## Funcionalidades
- Login de usuário
- Dashboard com dados do veículo
- Alertas inteligentes
- Histórico de manutenção
- Agendamento de revisão
- Tela de insights IA
- Integração com API REST

## Como executar

